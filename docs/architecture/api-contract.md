## Base
- **URL**: `http://localhost:4000/api` (dev) · `https://task-manager.vercel.app/api` (prod)
- **Auth**: JWT Bearer token en header `Authorization`
- **API Key**: Header `x-api-key` requerido en **todos** los endpoints (gate global)
- **Content-Type**: `application/json`
- **Errors**: Formato estándar `{ "error": "message" }`

### Error global
| Code | Message | Causa |
|---|---|---|
| 403 | API key inválida | Header `x-api-key` ausente o incorrecto |

## Endpoints

### Auth

#### POST /api/auth/login
- **REQ**: REQ-005
- **Auth**: API key requerida, JWT no requerido
- **Request**:
  ```json
  { "username": "string", "password": "string" }
  ```
- **Response 200**:
  ```json
  {
    "token": "eyJhbGciOiJIUzI1NiIs...",
    "user": { "id": "uuid", "username": "string" }
  }
  ```
- **Errors**:
  | Code | Message | Causa |
  |---|---|---|
  | 403 | API key inválida | Header x-api-key ausente o incorrecto |
  | 401 | Credenciales inválidas | Username o password incorrecto |
  | 400 | Username y password son requeridos | Campos vacíos |
- **curl**:
  ```bash
  curl -X POST http://localhost:4000/api/auth/login \
    -H "x-api-key: {API_KEY}" \
    -H "Content-Type: application/json" \
    -d '{"username":"admin","password":"password123"}'
  ```

### Tasks

#### GET /api/tasks
- **REQ**: REQ-002
- **Auth**: JWT requerido
- **Request**: Headers: `Authorization: Bearer {token}`
- **Query Params**: ninguno (MVP sin paginación)
- **Response 200**:
  ```json
  {
    "tasks": [
      {
        "id": "uuid",
        "title": "string",
        "description": "string|null",
        "completed": false,
        "createdAt": "2026-07-08T12:00:00Z"
      }
    ]
  }
  ```
- **Ordering**: `created_at DESC` (más recientes primero)
- **Errors**:
  | Code | Message | Causa |
  |---|---|---|
  | 401 | No autenticado | Token ausente o inválido |
- **curl**:
  ```bash
  curl http://localhost:4000/api/tasks \
    -H "Authorization: Bearer {token}"
  ```

---

#### POST /api/tasks
- **REQ**: REQ-001
- **Auth**: JWT requerido
- **Request**:
  ```json
  { "title": "string", "description": "string (optional)" }
  ```
- **Validation**: `title` no vacío, trim whitespace
- **Response 201**:
  ```json
  {
    "id": "uuid",
    "title": "string",
    "description": "string|null",
    "completed": false,
    "createdAt": "2026-07-08T12:00:00Z"
  }
  ```
- **Errors**:
  | Code | Message | Causa |
  |---|---|---|
  | 400 | Título es requerido | Campo vacío o solo espacios |
  | 401 | No autenticado | Token ausente o inválido |
- **curl**:
  ```bash
  curl -X POST http://localhost:4000/api/tasks \
    -H "Authorization: Bearer {token}" \
    -H "Content-Type: application/json" \
    -d '{"title":"Comprar leche","description":"En el supermercado"}'
  ```

---

#### PATCH /api/tasks/:id
- **REQ**: REQ-003
- **Auth**: JWT requerido
- **Request**:
  ```json
  { "completed": true }
  ```
- **Response 200**:
  ```json
  {
    "id": "uuid",
    "title": "string",
    "description": "string|null",
    "completed": true,
    "createdAt": "2026-07-08T12:00:00Z"
  }
  ```
- **Errors**:
  | Code | Message | Causa |
  |---|---|---|
  | 404 | Tarea no encontrada | ID inexistente o de otro usuario |
  | 401 | No autenticado | Token ausente o inválido |
- **curl**:
  ```bash
  curl -X PATCH http://localhost:4000/api/tasks/{id} \
    -H "Authorization: Bearer {token}" \
    -H "Content-Type: application/json" \
    -d '{"completed":true}'
  ```

---

#### DELETE /api/tasks/:id
- **REQ**: REQ-004
- **Auth**: JWT requerido
- **Request**: Headers: `Authorization: Bearer {token}`
- **Response 204**: No content
- **Errors**:
  | Code | Message | Causa |
  |---|---|---|
  | 404 | Tarea no encontrada | ID inexistente o de otro usuario |
  | 401 | No autenticado | Token ausente o inválido |
- **curl**:
  ```bash
  curl -X DELETE http://localhost:4000/api/tasks/{id} \
    -H "Authorization: Bearer {token}"
  ```

## Field → DB Mapping
| API Field | DB Column | Transform |
|---|---|---|
| id | id | UUID directo |
| title | title | Trim whitespace |
| description | description | Nullable |
| completed | completed | Boolean |
| createdAt | created_at | snake_case → camelCase |
| username | username | Trim + lowercase |
| password | password_hash | bcrypt hash |

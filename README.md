# task-api

API RESTful escalable para la gestión avanzada de tareas (To-Do list). Además de las operaciones básicas (CRUD), la API soportará autenticación y autorización de usuarios, categorización de tareas, asignación de niveles de prioridad y fechas de vencimiento, y flujo de estados (Pendiente, En Progreso, Completada).

## Purpose

Proveer un backend robusto, seguro y altamente disponible que permita a individuos y equipos organizar su carga de trabajo, reducir la carga cognitiva y mejorar la productividad. Actuará como la fuente de verdad centralizada para futuras interfaces (web, móvil, escritorio).

## Getting Started

This project follows the [SDD Standard](https://github.com/FactoriaUnipago/unipago-sdd-standard).

### Setup
```bash
bash sdd-sync.sh --role analyst
```
